The objective of this project is to analyze customer data and build a machine learning model that can predict whether a customer will churn (leave the bank) or stay. The goal is to help understand key factors influencing churn and improve customer retention strategies. For Data Preprocessing, handled missing values and duplicate records, encoded categorical variables (Gender, Geography) and scaled features for better model performance. For Exploratory Data Analysis (EDA), analyzed distribution of the target variable (Churn), identified class imbalance using visualizations and observed patterns in customer behavior. For Model Building, trained two classification models: Logistic Regression and Decision Tree Classifier, split data into training and testing sets (80/20). For Model Evaluation, evaluated models using accuracy score and compared performance of both models. For Feature Importance Analysis, identified key factors influencing churn using: Decision Tree feature importance and Logistic Regression coefficients. Results and insights revealed that Logistic Regression achieved the highest accuracy of 81.10%, outperforming Decision Tree (78.05%). Feature importance analysis showed that certain customer attributes strongly influence churn behavior. Logistic Regression performed better due to better generalization on scaled data. The analysis helps identify important factors that can be used for customer retention strategies. In conclusion, This project demonstrates how machine learning can effectively predict customer churn. Logistic Regression proved to be the most suitable model for this dataset, and feature analysis provided valuable business insights for reducing customer loss.
